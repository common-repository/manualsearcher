== Description ==
Adds the ManualSearcher widget. Allows users to search user manuals from our database in a language of your choice.

= Widget settings =

1.	Choose the source to search for manuals. Default: UK.
2.	Checkbox for displaying logo or text. 
3.	Check the box <em>add link to website</em> to add a link to the website if you wish to do so.
4.	Replacement title for logo.
4.	Specify the width of the widget either in pixels or percentage. Default: 100%.
5.	Change the radius for more or less rounded corners. Default: 10px.
6.	Change the placeholder. Default: Enter Search Term. 
7.	Specify the style of the border. Default: 1px solid #a8a8a8.
8.	Add a background color if you want. Default: none.
9.	Add a description to the widget. Default: none.


== Installation ==

1.	Upload the plugin files to the <em>/wp-content/plugins/ManualSearcher</em> directory, or install the plugin through the WordPress plugins screen directly.
2.	Activate the plugin through the <em>Plugins</em> screen in WordPress
3.	Go to Appearance->Widgets and add the ManualSearcher widget from the <em>Available Widgets</em> to the widget area of your choose.
